# CTF Webserver

A CTF challenge webserver virtual machine.

## Download VM

Download the OVA file from Google Drive:
[Download CTF Webserver OVA]([https://drive.google.com/your-link-here](https://drive.google.com/file/d/1ER3W90N5FZjdpsYQD517-CmrfHpkB85d/view?usp=sharing))

## How to Import

1. Download and install [VirtualBox](https://www.virtualbox.org/)
2. Open VirtualBox → File → Import Appliance
3. Select the downloaded `.ova` file
4. Click Import and wait
5. Start the VM
